# Школа 71 - Система учёта школьного питания

Веб-приложение для учёта питания МБУ «Школа № 71».

## Роли
- **Учитель** — загружает табели питания своего класса
- **Администратор** — видит сводную заявку, статусы, расхождения, экспорт, email

## Стек
- **Backend**: Python 3.11 + FastAPI + SQLAlchemy + SQLite
- **Frontend**: React 18 + TypeScript + Vite
- **Excel**: xlrd (чтение .xls), xlsxwriter (экспорт .xlsx)
- **Auth**: JWT (python-jose + bcrypt)

## Быстрый старт

### Backend
```bash
cd backend
pip install -r requirements.txt
python seed.py        # Создаёт БД, классы, льготы, admin/teacher аккаунты
python run.py         # Запуск на http://localhost:8000
```

### Frontend
```bash
cd frontend
npm install
npm run dev           # Запуск на http://localhost:3000 (проксирует /api → :8000)
```

## Функционал
1. Загрузка табелей (.xls/.xlsx) учителем
2. Автоматический парсинг (Авангард/Любава, льготы СВО/МН/ОВЗ)
3. Автоматическое заполнение сводной заявки на питание
4. Сравнение с предыдущим днём и подсветка расхождений
5. Ручная редактура администратором (с аудитом)
6. Статусы «прислали / не прислали» по каждому классу
7. Месячное планирование
8. Экспорт в Excel (по дате / за месяц)
9. Формирование ZIP-архивов (заявка + табели)
10. Отправка по email
